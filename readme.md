# Obstacle Avoidance Car 🚗

A robotics project developed for the Semester 5 Robotics Course. This robot uses ultrasonic sensors to detect obstacles and navigate autonomously.

## 🛠 Features
* Autonomous navigation using HC-SR04 sensors.
* Custom 3D printed chassis (STL files included in `DESIGN/`).
* Dual-motor control logic (Arduino/ESP32).

## 📁 Project Structure
* `MOBILcode.ino`: Primary source code for the robot.
* `DESIGN/`: Contains `.stl` and `.3mf` files for 3D printing.
* `REPORT/`: Documentation and test results (PDF/Docx).
